# IP Address
- Also known as logical address 
- IP stands for "Internet Protocol"
- It is represented in Decimal 
- Example include IPV6 and IPv4
## Octets
- IP address is Represented in format *W.X.Y.Z* -> 172.30.10.11
	- These are known as the four octets
		- Each Octet can have a value of 0 - 255
		- Each Octet has 8 bits
## IPV4 and IPV6
Ipv4 is 32 bits while ipv6 is 128 bits

# Mac Address
- Also known as physical address, hardware address
- Represented in hex
- is 48 bits
- What is the company identifier and serial number in mac address

# Network
There are several logical layers developed for communication between 2 end points.  There are in total 7 layers abbreviated as **APSTNDP** .

**OSI MODEL**(Basic Explanation)

| **Numbers** | **Layers**         | **Example**                                                  |
| ----------- | ------------------ | ------------------------------------------------------------ |
| 7           | Application Layers | Browser                                                      |
| 6           | Presentation Layer | Transform the data into something such as encode it etc      |
| 5           | Session Layer      | Create a session with provider such as google                |
| 4           | Transport LAyer    | Send data through netword                                    |
| 3           | Network Layer      | Fix the errors etc, *IP addressing is part of network layer* |
| 2           | Data Link Layer    | *Mac address is part of DATA link layer*                     |
| 1           | Physical Layer     | The physical cable                                           |
