# IPV4

- IPV4 is 32 bits and is split into two parts
  - The parts are network parts and Host parts
  - The most significant is the network part
  - After deducing the network we can determine the host part
  - IN the before times, **class based** ip addressing was used
  - Now **classless interdomain routing** (CIDR) is used
